# 🎭 Escape Room – Mafia Noir 1950s

## 🎨 Dirección General

* **Mood:** Noir elegante, clandestino, opresivo pero sofisticado

* **Paleta global:**

  * Marrones oscuros / nogal
  * Rojos profundos / burdeos
  * Dorados envejecidos
  * Negro / carbón
  * Verde oscuro (puntual)

* **Iluminación global:**

  * Luz cálida contrastada con sombras frías

---

# 🧩 Zonas del Escape Room

## 1. Sala de Interrogatorio

### Materiales

* Hormigón o ladrillo pintado envejecido
* Metal industrial
* Madera barata / desgastada

### Props Obligatorios

* Silla metálica central
* Mesa de interrogatorio
* Foco colgante
* Cubo / trapo / herramientas
* Manchas / suciedad / humedad
